# Changelog

All notable changes to the SwiftUI Expert Skill will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-21

### Added
- Initial release of SwiftUI Expert Skill
- Workflow decision tree for review, improve, and implement tasks
- **State Management** reference — property wrapper selection guide (`@State`, `@Binding`, `@StateObject`, `@ObservedObject`, `@Environment`, `@Observable`)
- **View Composition** reference — Manferdini extraction rules, computed properties vs sub-views, `@ViewBuilder`
- **Modern APIs** reference — iOS 18/26 APIs, deprecation replacements, `#Preview`, `navigationDestination`
- **Performance Patterns** reference — `equatable`, lazy loading, `task(id:)`, `GeometryReader` alternatives
- **List Patterns** reference — stable identity, `ForEach` with `Identifiable`, `LazyVStack` trade-offs
- **Navigation Patterns** reference — `NavigationStack`, programmatic navigation, deep links
- **Animation Basics** reference — `withAnimation`, implicit/explicit animations, timing curves
- **Animation Transitions** reference — `AnyTransition`, combined transitions, matched geometry effect
- **Animation Advanced** reference — `PhaseAnimator`, `KeyframeAnimator`, custom `Animatable`
- **Scroll Patterns** reference — `ScrollView`, `scrollPosition`, `safeAreaPadding`, `scrollTargetBehavior`
- **Text Formatting** reference — `AttributedString`, `Text` interpolation, `textRenderer`
- **Image Optimization** reference — `AsyncImage`, `resizable`, `contentMode`, caching strategies
- **Liquid Glass** reference — iOS 26 `glassEffect`, `GlassEffectContainer`, correct usage rules
- 13 reference documents covering the full SwiftUI best-practices surface
- Complete documentation (README, INSTALLATION, QUICKSTART)
